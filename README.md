🚨 Crime Data Analysis Dashboard (2024–2025) – Power BI
This project presents an interactive Power BI dashboard that analyzes crime data from 2024 to 2025 across major Indian cities. The goal is to provide a visual summary that supports law enforcement and public safety officials in making faster, data-driven decisions.

📊 Dashboard Features:
Total Cases: 16,000+

Victim Age Range: 10 to 79 (Average Age: 44)

Crime Categorization: By domain, weapon used, gender, and day

Top 5 Cities by Case Volume

Daily Trends of Reported Incidents

Filter Options: City, date of occurrence, crime type, gender, case status, and more

💡 Key Benefits:

 Instead of manually checking thousands of case files, this dashboard helps law enforcement officials:

✅ Save valuable time by visualizing crime trends in seconds
✅ Identify high-risk areas and crime types instantly
✅ Act faster with insights filtered by gender, city, weapon type, and day of occurrence
✅ Make data-driven decisions to deploy resources effectively and prevent repeat crimes
Reduces time spent going through individual case files

Enables quick pattern recognition for proactive policing

Helps in resource optimization and faster incident response

Provides an intuitive interface for non-technical users

Power BI (Bar Charts, KPIs, Slicers, Tooltips, Filters)
